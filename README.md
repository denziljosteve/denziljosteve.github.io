# 🌐 Denzil Josteve Fernandes — Personal Portfolio

A modern, dark-luxury personal portfolio website built to showcase my experience in **Digital & AI Strategy, Product Management, and Consulting**.

🔗 **Live Site:** [denziljosteve.github.io](https://denziljosteve.github.io)

---

## ✨ Features

- **Dark luxury design** — obsidian black with warm gold accents
- **Fully responsive** — optimized for desktop, tablet, and mobile
- **Smooth animations** — scroll fade-ins, hero reveal, parallax orbs, and metric counters
- **Sections:** Hero · About · Experience · Education · Projects · Skills · Leadership · Achievements · Contact
- **Frosted glass navbar** that activates on scroll with active section highlighting
- **One-click CV download** linked directly to resume PDF

---

## 🗂️ File Structure

```
portfolio/
├── index.html                        # Main HTML file
├── style.css                         # All styles (CSS variables, layout, animations)
├── script.js                         # Interactions (scroll, nav, parallax, counters)
├── Denzil Josteve Fernandes.png      # Profile photo
└── Denzil Josteve Fernandes_CV.pdf   # Resume (linked via Download CV button)
```

---

## 🛠️ Built With

- **HTML5** — semantic, clean markup
- **CSS3** — custom properties, grid, flexbox, keyframe animations
- **Vanilla JavaScript** — IntersectionObserver, parallax, counter animations
- **Google Fonts** — Playfair Display + DM Sans
- **Lucide Icons** — lightweight SVG icon set

No frameworks. No build tools. No dependencies to install — just open `index.html`.

---

## 🚀 Running Locally

```bash
git clone https://github.com/denziljosteve/portfolio.git
cd portfolio
# Open index.html in your browser
open index.html
```
